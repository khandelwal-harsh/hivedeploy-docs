## How to try endpoints

To execute requests from this page, you need an API token.

<Callout type="info">
A dedicated API tokens UI is on the roadmap. For now, copy a session JWT from your browser's DevTools after logging in at `app.hivedeploy.in`:

```js
localStorage.getItem('auth_token')
```

Paste it into Scalar's Auth panel below. Tokens are stored only in `sessionStorage` and are wiped when you close the tab.
</Callout>
